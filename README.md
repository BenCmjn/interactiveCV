# Interactive CV - Benjamin Calméjane

An interactive résumé built as a single HTML file. No framework, no build step.

## Features

- Filter experiences by category (B2C, B2B, Design System, Entrepreneurial)
- Smooth animated transitions on filter changes
- Responsive layout (mobile-friendly)
- Zero dependencies — one `index.html`, Google Fonts via CDN

## Usage

Just open `index.html` in a browser. No server required.
